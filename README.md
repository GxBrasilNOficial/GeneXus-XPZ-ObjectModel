# Knowledge-GeneXus

Conhecimento compartilhado sobre GeneXus, com foco em pesquisa prática, convenções operacionais e notas reutilizáveis para análise de objetos e pacotes `XPZ`.

## Conteúdo

- `genexus-xpz-research.md`: descobertas práticas sobre a estrutura de exportações `XPZ`
- `genexus-xpz-generation-rules.md`: regras operacionais para geração, nomeação e organização de exportações `XPZ`
- `genexus-object-design-patterns.md`: padrões e observações de modelagem de objetos
- `genexus-open-points.md`: pontos em aberto e trilhas de investigação

## Objetivo

Disponibilizar material útil para a comunidade GeneXus, especialmente para análise de Knowledge Bases, exportações `XPZ` e organização de artefatos técnicos.
